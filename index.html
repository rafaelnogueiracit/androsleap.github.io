The "Micro-Agent" Strategy

To ensure accuracy, we do not feed the entire rulebook into one AI prompt. We split the 9 Validation Rules into 4 specialized agents.

üÜî

The Identity Agent

"Who is this and are documents consistent?"

  • Rule 1: NPI (10-digit) & TIN Format Check.
  • Rule 8: Document Freshness (< 180 days).
  • Rule 9: Missing Document Check.
üìú

The License Agent

"Is this person legally allowed to practice?"

  • Rule 2: State License Expiration & Match.
  • Rule 6: DEA Certificate Active Status.
  • Rule 7: Board Certification Verification.
üìÖ

The Career Agent

"Does their work history make sense?"

  • Rule 4: Work History Gap Analysis (> 6 months).
  • *Uses Nova Pro to read unstructured CVs and find explanations for gaps (e.g., "Sabbatical").
üö´

The Risk Agent

"Is this person on a blacklist?"

  • Rule 3: Malpractice Insurance Limits ($1M/$3M).
  • Rule 5: Sanctions Check (OIG/SAM/OFAC via Lambda).

Complexity vs. Risk Analysis

Why we need Amazon Nova Pro: Simple models fail when file complexity (Pages) and Risk Factors (Gaps/Sanctions) increase.
